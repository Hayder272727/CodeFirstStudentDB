# Code First Student Database

## Description
This is a simple C# Console Application using Entity Framework Code-First approach.
The project automatically creates a Student database and inserts one student record.

## What it does
- Uses Entity Framework Code First.
- Creates a Student table in SQL Server LocalDB.
- Adds one student to the database.
- Saves the data automatically.

## How to run
1. Download or clone the repository.
2. Open the solution in Visual Studio.
3. Restore NuGet packages if required.
4. Press F5 to run the application.
5. The database will be created automatically.

## Technologies Used
- C#
- Entity Framework
- SQL Server LocalDB
- Visual Studio
